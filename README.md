# Pine
<br/>A Compiled Programming Language written in C++. 

# Abstract
Pine is a compiled programming language that's intended for 64-bit machines. Pine depends on `gcc`. Pine is a statically-typed programming language with syntax similar to Swift. A more comprehensive Pine tutorial will be written in the future.
